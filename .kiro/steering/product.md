# Product Overview

## Application Purpose
A fullstack todo application demonstrating AWS Amplify Gen 2 integration with Cloudscape Design System.

## Core Features
- User authentication with Amazon Cognito
- Todo item management (create, read, update, delete)
- Owner-based data isolation (users only see their own todos)
- Responsive UI built with Cloudscape Design System

## Target Users
- Developers learning AWS Amplify Gen 2
- Teams needing a reference implementation for Amplify + React projects
- Users looking for a production-ready todo application template

## Key Value Propositions
1. **Fullstack Integration**: Complete example connecting React frontend to AWS backend services
2. **Best Practices**: Implements AWS Amplify Gen 2 patterns and Cloudscape UI guidelines
3. **Deployment Ready**: Configured for AWS Amplify Hosting with CI/CD
4. **Testing Included**: Comprehensive test setup with Vitest and React Testing Library
